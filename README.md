  - Technical test for nata.house

- ## Stack

  - React
  - Monorepo
  - Docker

- ## Running

  - In root folder, execute command: `docker-compose up --build`

- ## Project

  - After run docker command, wait for build and [Client](http://localhost:3000) will be avaiable.
  - Client uses port 3000.
